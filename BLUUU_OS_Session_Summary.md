# 1. КОНЦЕПЦИЯ BLUUU OS

## Что это
BLUUU OS — операционная AI-система для масштабирования Bluuu Tours в новые локации. Вместо классической франшизы, где знания передаются через людей, BLUUU OS упаковывает операционное know-how в AI-систему, которая разворачивается вместе с лодками Fortune Yachts и одним управляющим.

## Вертикальная интеграция
- **Fortune Yachts** → производство и доставка лодок
- **BLUUU OS** → AI-операционная система, автоматизация процессов
- **Bluuu Tours** → бренд, стандарты, 8 лет опыта, 9,000+ отзывов

## Конкурентное преимущество (moat)
Ни один конкурент не объединяет производство лодок + проверенную операционную модель + AI-систему управления. Для копирования нужно воссоздать все три компонента с нуля.

## Аналоги в мире
- **Breezeway** — ближайший по духу: "Smart Operating System powered by AI" для vacation rentals, 265,000+ объектов в 85 странах. Но для аренды, не для boat tours.
- **Carpe Diem Tours** — тур-оператор в 7 городах Европы, активно использует AI (Claude, Perplexity). Начали так же — "список всех задач каждой команды → автоматизация".
- **Tourbase** — франшиза в туризме ($35k взнос), но модель посредника, не владеют лодками.
- **Прямого клона BLUUU OS не существует.**

# 5. GUIDE RACE — СИСТЕМА МОТИВАЦИИ

## Философия
Мотивация распределена на весь экипаж, не только гидов. Два уровня соревнования: личный (Guide Race) и командный (Boat Race).

## Очковая система

### Личные очки гида
| Действие | Очки | Деньги |
|----------|------|--------|
| 5★ отзыв с именем | +10 | +IDR 100K |
| 4★ отзыв с именем | +5 | — |
| Развёрнутый комплимент (AI) | +5 bonus | — |
| Check-in вовремя (день) | +1 | — |
| Полный месяц без опозданий | +15 bonus | — |
| Менторство новичка | +20 | +IDR 200K |
| Негативный отзыв (2★ и ниже) | -15 | — |

### Командные очки лодки
| Действие | Очки |
|----------|------|
| Каждый 5★ отзыв по туру лодки | +5 team pts |
| Ноль жалоб за неделю | +10 team pts |

## Уровни гидов
| Уровень | Очки/мес | Бонус/мес | Привилегии |
|---------|----------|-----------|-----------|
| **ROOKIE** | Первые 2 мес | Базовая ставка | Всегда в паре с Silver+. Обучение |
| **BRONZE** | 0-50 | 100K/review | Стандартные Classic группы |
| **SILVER** | 51-100 | +300K/мес | Classic + иногда Premium. Приоритет выходных |
| **GOLD** | 101-150 | +700K/мес | Premium группы. Менторство новичков (+200K) |
| **DIAMOND** | 151+ | +1.2M/мес | Premium/Elite. Первый выбор расписания. Стена славы |

## Призы
- **Гид месяца**: IDR 1,000,000 + фото на сайте + первый выбор расписания
- **Лодка месяца**: IDR 500,000 каждому (гид + капитан + помощник)
- **Капитан**: доп IDR 300,000 за месяц без технических проблем

## AI-расписание

### Жёсткие правила (не нарушаются)
- ≤6 pax = 1 гид
- 7+ pax = 2 гида
- Не больше 6 дней подряд без выходного
- Rookie всегда в паре с Silver+

### Мягкие правила (AI оптимизирует)
- Diamond/Gold → Premium/Elite группы
- Равномерная нагрузка (разница макс 2 тура/неделю)
- Языковой матчинг (русская группа → русскоговорящий гид)
- После негатива → 3 дня только в паре
- Ротация маршрутов (не больше 3 дней подряд один)

## AI Review Parsing — как работает
1. Сервис (Trustindex / API) собирает отзывы с: Airbnb, TripAdvisor, Klook, Google Maps, GYG, Viator
2. Каждые 4-6 часов новые отзывы → Supabase
3. AI (GPT/Claude) парсит: имя гида, рейтинг, sentiment, ключевые темы
4. Если имени нет → привязка по дате тура из манифеста
5. Отзыв ниже 4★ → моментальный алерт Geebran
6. Очки начисляются автоматически
